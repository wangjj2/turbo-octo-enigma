# turbo-octo-enigma

## Setup

cd into the project root and run `npm run start`

## Project Layout 

### Folder Structure

* 1 Repo, 1 folder for frontend, 1 folder for backend.

### Git

#### Branches

* `dev/...` - Development branch
* `master` - Work branch
* `stable` - QA branch
* `production` - Release branch

#### Tags

* Use semver numbering
